OpenAI Agents SDK v0.17.4: Realtime カスタムボイス対応とトレーシング・MCP 安定性の向上

2026-05-26

主要アップデート

🎙️

Realtime カスタムボイスオブジェクト対応

gpt-realtime-2 エージェントでカスタム音声設定をオブジェクトとして渡せるようになりました。ピッチやスピードなどの細かな調整が可能に。

🔒

MCP SSE トランスポートのセキュリティ強化

強化された HTTP クライアントデフォルトが MCP 通信に適用され、セキュアな通信が標準化されました。

🔧

欠損ファンクションツールのオプトインリカバリ

ツール未発見時のリカバリ処理をオプトインで利用可能。本番環境の耐障害性が向上します。

📊

トレーシング改善

FunctionSpanData の修正、__slots__ エントリ追加、関数・型のエクスポート改善により、デバッグ体験が向上。

🛡️

ModelBehaviorError の JSON ペイロードリダクト

エラーログから機密情報漏洩のリスクを低減。本番環境でのセキュリティが強化されました。

主な機能と改善

🎙️ カスタムボイス

RealtimeAgent で voice パラメータにオブジェクトを渡し、ID やピッチ、スピードなどの音声特性を細かく制御可能に。

🔒 MCP セキュリティ

SSE トランスポートに強化 HTTP クライアントデフォルトを適用。MCPListToolsItem 等の型を直接インポート可能に。

🔧 ツールリカバリ

欠損ファンクションツールのリカバリをオプトインで有効化。ツールが見つからない場合でも処理を継続。

📊 トレーシング安定化

FunctionSpanData の修正と __slots__ エントリ追加で、トレーシングの信頼性とパフォーマンスが向上。

🛡️ ペイロードリダクト

ModelBehaviorError 内の JSON ペイロードをリダクトし、ログ出力時の機密情報漏洩を防止。

📦 エクスポート改善

関数・型のエクスポートが改善され、IDE の補完機能やタイプチェックの精度が向上。

コードサンプル

Python Realtime カスタムボイス設定例 
from agents.realtime import RealtimeAgent, RealtimeSession

agent = RealtimeAgent(
    name="voice_assistant",
    model="gpt-realtime-2",
    voice={
        "id": "custom-voice-id",
        "settings": {
            "pitch": 1.0,
            "speed": 1.0,
        }
    },
    instructions="You are a helpful voice assistant.",
)

async with RealtimeSession(agent) as session:
    await session.start()

開発者への影響

🎙️ Realtime エージェント開発者

カスタムボイスオブジェクトを直接渡せるようになり、音声エージェントのカスタマイズ性が大幅に向上。ピッチやスピードの調整が容易になります。

🔌 MCP 統合利用者

セキュリティ強化に加え、MCPListToolsItem 等の型を直接インポートできるようになり、型安全な MCP 統合コードの記述が可能に。

🏢 本番環境運用者

JSON ペイロードリダクトによるログセキュリティの強化と、欠損ツールリカバリ機能により、本番環境での耐障害性が向上。
