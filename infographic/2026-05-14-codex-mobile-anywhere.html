コードサンプル
Python Codex タスク作成と WebSocket 監視 
import openai
import asyncio

client = openai.OpenAI()

# モバイルからタスクを作成
task = client.codex.tasks.create(
    description="ユーザー認証モジュールにOAuth 2.0サポートを追加",
    repository="github.com/myorg/myapp",
    branch="feature/oauth-support",
    notify=True  # モバイルプッシュ通知を有効化
)

print(f"Task ID: {task.id}")
print(f"Status: {task.status}")

# WebSocket でリアルタイム監視
async def monitor_task(task_id):
    async with client.codex.tasks.stream(task_id) as stream:
        async for event in stream:
            if event.type == "status_change":
                print(f"[Status] {event.status}")
            elif event.type == "approval_required":
                print(f"[Approval] {event.description}")
                # モバイルからワンタップ承認
                client.codex.tasks.approve(
                    task_id=task_id,
                    change_id=event.change_id
                )

asyncio.run(monitor_task(task.id))
ユースケース

通勤中のタスク承認

電車やバスでの移動中に、Codex が完了したコード変更をプッシュ通知で受け取り、差分を確認して承認。オフィスに着く前にデプロイメントパイプラインが進行し、チームの待ち時間をゼロに。

外出先からの緊急対応

本番環境で障害が発生した際、外出先からモバイルで Codex にホットフィックスタスクを依頼。リアルタイムでログを監視し、修正コードを即座に承認してデプロイ。デスクトップの前に戻る必要なし。

チーム間の非同期開発

異なるタイムゾーンで働くチームメンバーが、それぞれの活動時間帯にモバイルからレビューと承認を実施。24 時間途切れることなく開発が進行し、グローバルチームの生産性が飛躍的に向上。
