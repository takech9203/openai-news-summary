概要

🛡️

コミュニティ安全のための包括的フレームワークを発表

OpenAI がコミュニティの安全を確保するための多層的なアプローチを体系化。ポリシー、技術、パートナーシップを統合した包括的なフレームワークを公開した。

🔍

Moderation API とコンテンツフィルタリングの強化

omni-moderation-latest モデルを含む最新の Moderation API により、テキスト・画像のマルチモーダルなコンテンツモデレーションを提供。開発者が安全なアプリケーションを構築するための基盤を強化。

🤝

外部パートナーシップと透明性の向上

GASA (Global AI Safety Alliance)、各国の AI Safety Institute、People-First AI Fund など、外部機関との連携を通じてコミュニティ全体の安全性を向上。

🧪

Safety Bug Bounty と Bio Bug Bounty プログラムの継続

外部のセキュリティ研究者やドメインエキスパートと協力し、脆弱性の発見と修正を継続的に実施。報奨金プログラムを通じたオープンな安全性改善の取り組み。

📋

開発者向け安全ツールとガイドラインの拡充

API 利用ポリシー、安全ガイドライン、OpenAI Academy を通じて、開発者エコシステム全体の安全性リテラシーを向上させる取り組みを強化。

AI の安全性を科学的に追求する研究活動

  • アラインメント研究
  • レッドチーミング
  • モニタビリティ評価

開発者コミュニティ全体の安全性向上

  • OpenAI Academy による教育

業界・政府・市民社会との連携

Moderation API コードサンプル

omni-moderation-latest モデルを使用して、テキストコンテンツのモデレーションを実行するサンプルコード。カテゴリごとのスコアを確認し、閾値を超えたカテゴリをフラグとして検出する。

Python Moderation API の基本的な使い方 
from openai import OpenAI

client = OpenAI()

# テキストモデレーション
response = client.moderations.create(
    model="omni-moderation-latest",
    input="チェックするテキスト"
)

result = response.results[0]
print(f"フラグ: {result.flagged}")

for category, score in result.category_scores.model_dump().items():
    if score > 0.5:
        print(f"  {category}: {score:.3f}")

モデレーションカテゴリ

Moderation API が検出可能な主要カテゴリの一覧。各カテゴリに対して 0 から 1 のスコアが返され、閾値に基づいてフラグが設定される。

カテゴリ 説明
hate ヘイトスピーチ
harassment ハラスメント
self-harm 自傷行為
sexual 性的コンテンツ
violence 暴力
illicit 違法行為

ユースケース

🏢 プラットフォーム運営者

Moderation API を活用してユーザー生成コンテンツ (UGC) を自動フィルタリング。大規模なコンテンツモデレーションを効率的に実現し、安全なプラットフォーム運営を支援する。

💻 企業開発者

Instruction Hierarchy を活用してプロンプトインジェクション対策を実装。システムプロンプトとユーザー入力の優先順位を適切に制御し、意図しない動作を防止する。

🔬 研究者

Safety Bug Bounty プログラムを通じて脆弱性を報告し、報奨金を獲得。AI システムの安全性向上に直接貢献しながら、セキュリティ研究の成果を社会に還元する。

まとめ

📝

OpenAI のコミュニティ安全への取り組みは、コンテンツポリシー、モデレーションツール、安全研究、開発者エコシステム、外部パートナーシップの 5 つの柱で構成される包括的なフレームワークである。Moderation API や Instruction Hierarchy などの技術的ツールと、GASA や AI Safety Institute との外部連携を組み合わせることで、多層的な安全性を実現している。Safety Bug Bounty や Bio Bug Bounty プログラムを通じた外部コミュニティとの協働、OpenAI Academy を通じた開発者教育など、エコシステム全体の安全性リテラシー向上にも注力している点が特徴的である。
