API 更新 / 料金

コンテナセッション課金モデルの更新 - 分単位課金と 5 分最低課金の導入

2026-06-02

概要

💰

分単位課金への移行

従来の 20 分固定課金から、実際の使用時間に応じた分単位課金に変更。最低課金時間は 5 分。

📉

最大 75% のコスト削減

5 分間のセッションでは従来比 75% の料金削減。短時間タスクほど大きな恩恵を受ける。

⚡

単価は据え置き

分あたりの料金 (1 GB: $0.03、4 GB: $0.12、16 GB: $0.48、64 GB: $1.92) は変更なし。

🐳

対象: Shell、Code Interpreter、Skills

ホストされたコンテナセッション全般に適用。サンドボックス環境での AI エージェント実行が対象。

対象サービス

🖥️ Shell ツール

ホストされたコンテナ内でシェルコマンドを実行。パッケージインストール、テスト実行、デプロイ作業に最適。

🐍 Code Interpreter

Python コード実行環境。データ分析、可視化、ファイル処理をサンドボックス内で安全に実行。

🧩 Skills

バージョン管理されたスキルバンドルをアップロードし、ホストされたシェル環境で再利用可能。

コードサンプル

Python Shell ツールを使用したコンテナセッション 
from openai import OpenAI

client = OpenAI()

# Shell ツールを使用した Responses API 呼び出し
response = client.responses.create(
    model="gpt-4.1",
    tools=[
        {
            "type": "container",
            "container": {
                "environment": "shell",
                "memory_gb": 4,  # 4 GB メモリティア ($0.12/分)
            }
        }
    ],
    input="プロジェクトの依存関係をインストールしてテストを実行してください"
)

print(response.output_text)

# セッションは使用した分数のみ課金される (最低 5 分)
# 例: 7 分で完了 → $0.12 × 7 = $0.84
# 変更前: $0.12 × 20 = $2.40
Python Code Interpreter でデータ分析 
from openai import OpenAI

client = OpenAI()

# Code Interpreter でデータ分析を実行
response = client.responses.create(
    model="gpt-4.1",
    tools=[
        {
            "type": "container",
            "container": {
                "environment": "code_interpreter",
                "memory_gb": 1,  # 1 GB メモリティア ($0.03/分)
            }
        }
    ],
    input="添付した CSV ファイルの売上データを分析してグラフを作成してください"
)

# 5 分で完了した場合: $0.03 × 5 = $0.15 (変更前: $0.03 × 20 = $0.60)

開発者への影響

短時間シェルコマンド実行 (削減率: 65-75%)

AI エージェントが単純なコマンドを実行するケース。パッケージインストール、ファイル操作、環境確認など 5-7 分で完了するタスクで最大の恩恵。

コードレビュー・テスト実行 (削減率: 40-60%)

テストスイートの実行やコード品質チェックなど、8-12 分程度のセッション。CI/CD パイプラインの一部として利用する場合に効果的。

データ分析・可視化 (削減率: 25-50%)

Code Interpreter でのデータ処理やグラフ作成。10-15 分程度のセッションでも一定のコスト削減が得られる。

設計上の考慮事項

⏱️ 5 分最低課金

5 分未満で終了しても 5 分分の料金が発生。極端に短いタスクはセッション再利用を検討。

🔄 セッション設計の見直し

20 分固定前提でまとめていたタスクを、論理的な単位で分割することが経済的に合理的に。

🎯 トークン課金は別途

コンテナ内のツール使用に伴うトークンは選択したモデルの単価で別途課金。変更なし。
