SDK UPDATE ALPHA

Codex CLI v0.135.0-alpha: Python SDK Beta、メモリシステム刷新、Vim テキストオブジェクト対応

2026-05-27 | 95 commits from v0.134.0

概要

🐍

Python SDK Beta

Codex CLI の Python SDK が独立ベータリリースとして公開。Rust ツールチェインなしで Python プロジェクトに統合可能に。

🧠

メモリシステムの全面刷新

メモリ状態を専用 SQLite DB に移行。アドホックメモリノート、エクステンション化されたプロンプトビルダー、メトリクス追跡を実装。

⌨️

Vim テキストオブジェクトバインディング

完全な Vim テキストオブジェクト操作を TUI に実装。word-end / line-end 動作、設定可能なターン中断キーバインド対応。

📊

Markdown テーブルレンダリング強化

アプリスタイルのテーブル描画と、狭小テーブルのキーバリューレコード変換に対応。OSC 8 ハイパーリンクも実装。

🔌

エクステンションシステムの拡充

スタンドアロン Web 検索、スレッドアイドルライフサイクルフック、Goal 使用量制限、Bedrock Namespace Tools を追加。

Before / After: メモリシステム

Before (v0.134.0)

  • ファイルベースのメモリ管理
  • MCP crate による分散ストレージ
  • 動的ツールストレージとの冗長性
  • メトリクスなし
  • アドホックなメモリ追加不可

After (v0.135.0-alpha)

  • 専用 SQLite DB による一元管理
  • エクステンションベースのプロンプトビルダー
  • 冗長ストレージの削除
  • メトリクス追跡による可観測性
  • アドホックメモリノートツール

主要新機能

🐍 Python SDK Beta

独立したベータリリースとして Python SDK を公開。

  • 独立パッケージメタデータ
  • フレンドリーなサンドボックスプリセット
  • CodexConfig リネーミング
  • ドキュメント整備

🧠 メモリシステム刷新

SQLite DB ベースの新メモリアーキテクチャ。

  • 専用 SQLite DB
  • アドホックメモリノート
  • プロンプトビルダーのエクステンション化
  • メモリツールコールメトリクス

⌨️ Vim テキストオブジェクト

完全な Vim テキストオブジェクトバインディングを実装。

  • iw, aw, i", a( など
  • word-end / line-end 動作
  • 設定可能なターン中断キーバインド

📊 Markdown テーブル

ターミナル内でのテーブル表示を大幅改善。

  • アプリスタイルのレンダリング
  • 狭小テーブルの KV レコード変換
  • ハイパーリンク対応

🔗 OSC 8 Web リンク

リッチコンテンツにクリック可能なハイパーリンクを追加。対応ターミナルで URL を直接クリックしてブラウザ表示。

🔍 Web 検索エクステンション

Web 検索機能をスタンドアロンエクステンションとして独立。有効/無効の柔軟な制御が可能に。

💤 スレッドアイドルフック

新ライフサイクルフック。アイドル状態時に自動保存やリソース解放を実行可能。

🎯 Goal Usage Limits

Goal エクステンションに使用量制限を実装。タスク実行コストの制御と予期せぬリソース消費を防止。

🏥 環境診断 (codex doctor)

環境診断機能を追加。システム環境の問題を自動検出し、トラブルシューティングを支援。

☁️ Bedrock Namespace Tools

AWS Bedrock 向けネームスペースツールを有効化。Bedrock 経由のツール統合を改善。

codex doctor

Python Python SDK Beta の使用例 (概念的) 
from codex import CodexConfig, CodexClient

# 設定の初期化 (CodexConfig にリネーム済み)
config = CodexConfig(
    sandbox_preset="friendly",  # フレンドリーなサンドボックスプリセット
    memory_enabled=True,
)

# クライアントの作成
client = CodexClient(config=config)

# タスクの実行
result = client.run("Refactor the authentication module")
print(result.output)

開発者への影響

Python 開発者への新たな選択肢

Python SDK Beta により、Rust ツールチェインなしで Codex CLI の機能を Python プロジェクトに統合可能。フレンドリーなサンドボックスプリセットでセットアップも簡素化。

メモリ管理の再設計による影響

SQLite への移行に伴い既存メモリデータのマイグレーションが必要な可能性あり。アドホックメモリノートで知識蓄積がより明示的に制御可能に。メトリクスでメモリ使用パターンの最適化が可能。

Vim ユーザーの生産性向上

フルテキストオブジェクトバインディングにより、Vim ユーザーが慣れた操作体系で TUI を利用可能。設定可能なキーバインドで個人の好みに合わせたカスタマイズが容易。

ターミナル表示の改善

Markdown テーブルの美しいレンダリング、OSC 8 ハイパーリンクによるワンクリック URL 表示、狭いターミナルでのキーバリュー形式表示により視認性が向上。

アルファ版利用時の注意

本リリースはアルファ版であり、API の破壊的変更が発生する可能性あり。本番環境での使用は推奨されない。フィードバックは GitHub Issues で受付中。
