TanStack npm サプライチェーン攻撃 "Mini Shai-Hulud" への対応

2026-05-13 | OpenAI News/Blog

URGENT: macOS アプリ強制アップデート期限

2026 年 6 月 12 日

すべての macOS ユーザーは、上記の期限までに ChatGPT for Mac およびその他の OpenAI macOS アプリケーションを最新バージョンに更新する必要があります。期限後は旧バージョンのアプリが起動不可になります。

Key Points

🚨

TanStack npm パッケージへのサプライチェーン攻撃

数百万の開発者が利用する TanStack エコシステム内の npm パッケージが侵害され、悪意のあるコードが正規バージョンとして公開された

🐛

"Mini Shai-Hulud" マルウェア

CI/CD 環境でのみ動作する標的型マルウェア。ビルド環境の認証情報、署名キー、環境変数等の機密情報を外部に送信する

🔐

コード署名証明書への影響

OpenAI のビルドパイプラインが侵害され、コード署名証明書のセキュリティに影響。即時ローテーションと HSM ベースの署名に移行

⏰

macOS アプリ更新期限: 2026 年 6 月 12 日

期限後は旧証明書で署名されたアプリが macOS Gatekeeper により実行拒否される。自動更新の有効化を確認すること

🛡

多層防御の強化

依存関係の integrity hash 検証、ビルド環境の零信頼化、SBOM 自動生成、エアギャップ署名環境など包括的な防御策を導入

攻撃の特徴 (Mini Shai-Hulud)

🕶 潜伏性

正規のパッケージコードに紛れ込み、通常の静的解析では検出困難。開発者のローカル環境では無害に動作する

🌐 拡散範囲

依存関係チェーンを通じて間接的に多数のプロジェクトに波及。TanStack は数百万の週間ダウンロード数を誇る

🎯 標的型動作

特定の環境変数 (CI、CODE_SIGNING_KEY 等) やビルド環境を検出した場合にのみ悪意のある動作を実行する

📤 情報窃取

ビルド環境の認証情報、署名キー、環境変数等の機密情報を外部サーバーに送信する

セキュリティ対策の比較

BEFORE - 従来のセキュリティ

  • ビルドサーバー上のキーチェーンに証明書保管
  • ビルドパイプライン内でコード署名を実行
  • CI/CD サービスアカウントによるアクセス制御
  • ログベースの事後確認による監査
  • 手動による証明書失効対応

AFTER - 強化後の防御策

  • HSM (Hardware Security Module) に証明書保管
  • エアギャップ環境で分離してコード署名
  • 多要素認証 + 承認ワークフロー
  • リアルタイム監視 + アラート
  • 自動検知 + 即時ローテーション

サプライチェーン防御の強化策

依存関係のロックダウン

すべての npm パッケージに対して完全な integrity hash の検証を導入し、想定外のパッケージ変更を検出

ビルド環境の零信頼化

ビルドパイプラインのネットワークアクセスをデフォルト拒否に設定し、許可されたレジストリのみへの通信を許可

SBOM (Software Bill of Materials) の自動生成

すべてのビルドアーティファクトに対して機械可読な SBOM を生成し、依存関係の透明性を確保

リアルタイム依存関係監視

npm パッケージのバージョン変更をリアルタイムで監視し、異常なパブリッシュパターンを検出してアラートを生成

サンドボックスビルド環境

Running Codex safely で解説されたサンドボックス技術をビルドパイプラインにも適用し、マルウェアの実行影響を封じ込め

署名プロセスのエアギャップ化

コード署名を完全に隔離された環境で実施し、ビルドパイプラインの侵害が署名証明書に波及しないアーキテクチャを採用

macOS アプリセキュリティ要件

🛡 Gatekeeper 検証

macOS は起動時にアプリケーションの署名を Gatekeeper で検証し、失効した証明書で署名されたアプリの実行を拒否する

🔍 OCSP チェック

Online Certificate Status Protocol により、証明書の有効性がリアルタイムで確認される

✅ Notarization 必須化

Apple のノータリゼーション (公証) を通過したアプリケーションのみが Gatekeeper を通過可能

🔄 自動更新メカニズム

OpenAI アプリの自動更新機能により、期限前に新バージョンへの移行を促進

⚠ ACTION REQUIRED: 開発者への必須対応

強制アップデート期限
2026 年 6 月 12 日

macOS ユーザー

  • ChatGPT for Mac およびその他の OpenAI macOS アプリケーションを最新バージョンに更新する
  • macOS の「システム設定 > App Store > 自動アップデート」が有効であることを確認する
  • アプリケーションメニューの「ChatGPT について」から、2026 年 5 月以降にリリースされたバージョンであることを確認する
  • 期限後は旧バージョンが起動不可になるため、事前に更新を完了すること

npm パッケージ利用者

  • プロジェクトで @tanstack/react-query、@tanstack/router 等を使用している場合、影響を受けたバージョンを確認する
  • package-lock.json 内の TanStack 関連パッケージの integrity hash が正規のものと一致するか検証する
  • npm audit コマンドで既知の脆弱性が報告されていないか確認する
  • CI/CD パイプラインで機密情報が環境変数として露出していないか確認し、シークレット管理ツールへの移行を検討する

セキュリティ対策の推奨事項

  • package-lock.json や npm shrinkwrap を活用し、依存関係のバージョンを完全に固定する
  • npm の provenance 機能を利用して、パッケージのビルド元を検証する
  • --ignore-scripts オプションの使用や、allow-scripts 設定による明示的な許可リスト管理を検討する
  • Socket.dev 等のサプライチェーン監視ツールの導入を推奨する
  • ビルドプロセス中の外部通信を許可リスト方式で制限し、情報漏洩リスクを低減する
