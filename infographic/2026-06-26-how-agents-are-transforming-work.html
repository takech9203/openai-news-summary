概要

📈

AI エージェント利用が劇的に増加

2025 年 11 月〜2026 年 6 月の約 7 か月間で、企業における AI エージェントの利用が爆発的に成長。複数の業務領域で二桁倍の成長を記録。

🚀

業務領域別の成長率

リサーチ 56 倍、カスタマーサポート 32 倍、エンジニアリング 27 倍、法務 13 倍の成長を達成。全ての主要業務領域で顕著な拡大。

🔗

同日発表との関連

Broadcom Jalapeno チップとサムスン展開と同日発表。インフラ強化とエージェント利用拡大の相乗効果を示唆。

成長の主要因

🧠 モデル性能の向上

GPT-5 シリーズで複雑なマルチステップタスクの自律処理が向上。エージェントの信頼性と実用性が飛躍的に改善。

🔧 API の成熟

Assistants API / Responses API の成熟により、企業が独自エージェントを容易に開発可能に。

🏢 Enterprise/Team の普及

ChatGPT Enterprise/Team の普及で、組織全体での AI 導入が加速。部門横断的な活用が拡大。

💻 Codex の一般提供

コーディングエージェントがエンジニアリング組織に浸透。400 万+ WAU を達成し標準ツール化。

開発者への影響

エージェント構築スキルの需要増加

AI エージェントの急速な普及に伴い、エージェント設計・構築・運用のスキルセットが開発者に求められる時代に。Assistants API や Responses API を活用したエージェント開発が新たなキャリアパスとして確立。

AI コーディングエージェントの標準ツール化

Codex 400 万 WAU は、開発者コミュニティで AI コーディングエージェントが標準ツールとして定着したことを示す。コードレビュー、テスト生成、リファクタリングなど日常的な開発タスクにエージェントが組み込まれる。

OpenAI エコシステムの優位性

大規模利用実績データに基づくフィードバックループが形成。ユーザーの利用データがモデル改善に活用され、さらなる利用拡大を促進する好循環が確立されている。
