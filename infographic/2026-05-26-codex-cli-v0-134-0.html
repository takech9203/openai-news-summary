Before / After

Before

  • レガシー v1 プロファイルが複数箇所に分散
  • 会話履歴の検索機能なし
  • MCP 環境ルーティング未対応

After

  • --profile フラグで一本化管理
  • 全文検索対応 (case-insensitive)
  • MCP 環境別ルーティング実装

主な新機能

🔍 全文検索

codex search コマンドで過去の会話スレッドを横断的に検索。結果はプレビュー付きで即座に確認可能

👤 プロファイル v2

CLI、TUI、sandbox の全インターフェースで統一されたプロファイル管理。レガシー v1 からの自動マイグレーションガイダンス付き

🔌 MCP 環境ターゲティング

MCP サーバーごとに環境を指定可能。OAuth クライアント認証オプションにも対応

⚡ 並列実行

readOnlyHint を持つ MCP ツールを自動で並列実行。複数ソースからの情報収集を高速化

🔗 スキーマ圧縮

$ref/$defs を保持したまま大きなスキーマを自動圧縮。コネクタツールの互換性向上

🧩 拡張機能強化

会話履歴コンテキストをエクステンションに公開。フックにサブエージェント ID を追加

Bash Codex CLI v0.134.0 使用例 
# ローカル会話履歴の検索
codex search "API endpoint implementation"

# プロファイルを指定して起動
codex --profile production chat

# MCP サーバーの追加 (OAuth オプション付き)
codex mcp add my-server \
  --url https://mcp.example.com/stream \
  --env production \
  --oauth-client-id "client_abc123" \
  --oauth-scope "read write"

# curl によるインストール
curl -fsSL https://codex.openai.com/install.sh | sh
