chat-latest のスナップショットが最新バージョンに更新されました。前回 (5 月 28 日) に続く定期的な更新です。

動的エイリアス

ChatGPT で使用されている最新の Instant モデルを指す動的エイリアスとして機能します。固定バージョンではありません。

⚡

コード変更不要

model="chat-latest" を指定するだけで、常に最新モデルを自動的に追従できます。

🏢

本番環境では固定バージョン推奨

再現性が必要な本番環境では gpt-5.5 等の固定バージョンの使用が推奨されます。

コードサンプル

Python chat-latest の使用例 
from openai import OpenAI

client = OpenAI()

response = client.chat.completions.create(
    model="chat-latest",
    messages=[
        {"role": "user", "content": "Hello!"}
    ]
)

print(response.choices[0].message.content)
# response.model で実際のスナップショット名を確認可能
print(f"使用モデル: {response.model}")
