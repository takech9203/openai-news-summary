Introducing Operator — AI ブラウザエージェントの再始動

2026-06-20

概要

🤖

CUA モデルによる GUI 操作

GPT-4o のビジョン + 強化学習で画面上のボタン・メニューを自律操作

🔄

2025 年サービス停止 → 2026 年再始動

ChatGPT Agent 統合を経て独立プロダクトとして復活

🌐

ユニバーサルインターフェース

DOM セレクタ不要、視覚認識ベースでどんな Web サイトにも対応

⚡

Responses API 統合

computer ツールタイプで開発者がブラウザ自動化を API から利用可能

Before / After

Before: 従来の RPA

  • DOM セレクタ個別定義
  • UI 変更で破綻
  • 固定リトライロジック

After: Operator CUA

  • 視覚認識で自動適応
  • 自然言語指示
  • 文脈に応じた修復

主な機能

👁️

視覚認識

スクリーンショットから UI 要素を認識し、ボタン、テキストフィールド、メニューなどを自動的に検出

📐

空間推論

座標理解とクリック位置判断により、画面上の正確な操作ポイントを特定

🧩

マルチステップ計画

複雑なタスクを分解し、段階的に実行する計画能力を備えた自律エージェント

🛡️

安全機構

ユーザー確認、サンドボックス環境、操作ログによる透明性と安全性の確保

ユースケース

📝 Web フォーム自動入力・予約手続き

旅行予約、行政手続き、各種申込フォームなど、複数ステップにわたるフォーム入力を自然言語の指示だけで自動完了

🔍 マルチサイトにまたがる情報収集と比較

複数の EC サイトや情報ソースを横断的に閲覧し、価格比較やスペック比較を自動で実施

🔁 反復的なブラウザ業務の自律実行

日次レポートのダウンロード、定期的なデータ入力、ステータス確認など、繰り返し発生するブラウザ作業を自動化
