主な機能

💰 予算管理

組織全体、チーム単位、個人ユーザーに対して月次または四半期の予算上限を設定。ソフトブロック/ハードブロックを選択可能

🎛️ モデルティア制限

モデルごとに個別の利用上限を設定。高コストモデルの利用を制限しつつ、軽量モデルは無制限に利用可能

🔔 アラートシステム

カスタマイズ可能な閾値でアラートを発報。通知のみ、ソフトブロック、ハードブロックの 3 段階アクション

📊 ダッシュボード

リアルタイムの使用量追跡、トレンド分析、予算消化率グラフ。ドリルダウンで詳細確認

🔗 API / Webhook 連携

Admin API でポリシーをプログラム管理。Webhook で Slack、Teams、PagerDuty などに通知を送信

コードサンプル

Python Admin API 予算ポリシー作成 
from openai import OpenAI
client = OpenAI()

policy = client.organization.spend_controls.create(
    scope="organization",
    budget={"amount": 10000, "currency": "usd", "period": "monthly"},
    alerts=[
        {"threshold_percent": 75, "action": "notify"},
        {"threshold_percent": 90, "action": "notify"},
        {"threshold_percent": 100, "action": "soft_block"},
    ],
)

ユースケース

IT 予算計画

四半期予算を組織レベルで設定し、部門ごとのチーム予算に分配。月次レポートで実績を追跡し、予算超過を未然に防止

段階的ロールアウト

新モデル (GPT-5.5 など) の導入時にユーザーレベルで低い上限を設定。利用状況を観察しながら段階的に制限を緩和

FinOps 統合

Webhook と Admin API を活用し、既存の FinOps ツール (CloudHealth、Kubecost など) と連携。AI 支出を全体の IT コスト管理に統合
