OpenAI が AI ブラウザ Atlas を終了

2026-07-09

概要

🛑

Atlas (AI ブラウザ) の終了を決定

OpenAI はスタンドアロン AI ブラウザ「Atlas」の開発を終了し、既存製品へのブラウジング機能統合に戦略を転換

💬

「ブラウザは機能であり、目的地ではない」

独立したブラウザ製品ではなく、ユーザーが既に使っている環境にブラウジング機能を統合する方針を明確化

🧩

Chrome 拡張機能のローンチ

Google の Gemini サイドパネルと直接競合する Chrome 拡張機能をリリースし、ブラウザ内で ChatGPT にアクセス可能に

🖥️

デスクトップアプリに内蔵ブラウザ追加

ChatGPT デスクトップアプリにブラウザ機能を組み込み、サイト閲覧やファイル操作を AI と一体化

🔄

ChatGPT を「継続的なワークスペース」として再定義

チャットボットからワークスペースへの進化を加速し、ブラウジング、コード実行、ファイル管理を統合

Before / After

Before: Atlas スタンドアロンブラウザ

  • 独立した AI ブラウザ製品
  • ChatGPT とは別のアプリ
  • ユーザーの環境移行が必要
  • 単一チャネルでの提供

After: 統合ワークスペース

  • Chrome 拡張機能
  • デスクトップアプリ内蔵ブラウザ
  • クラウドブラウザ (エージェント用)
  • 既存環境に自然に統合

新しい 3 つの配信チャネル

🧩 Chrome 拡張機能

既存のブラウザ内で ChatGPT のパワーを活用

  • ページコンテキストの理解
  • 質問と対話
  • コンテンツ要約
  • タスク起動

🖥️ デスクトップアプリ

ChatGPT アプリ内でウェブブラウジングを完結

  • サイト閲覧
  • ログイン・アカウント操作
  • ファイルダウンロード
  • ページ操作・インタラクション

☁️ クラウドブラウザ

AI エージェントによる自律的なウェブ操作

  • AI エージェント駆動
  • 自律タスク実行
  • バックグラウンド処理
  • 複雑なワークフロー自動化

開発者への影響

Chrome 拡張機能 API の活用

Chrome 拡張機能を通じて ChatGPT の機能にアクセスする新しいインテグレーションパターンが登場。ウェブアプリ開発者は、ユーザーが ChatGPT 拡張機能と連携することを前提とした UX 設計が可能に

エージェント向けクラウドブラウザ

AI エージェントがクラウドブラウザを通じてウェブ上のタスクを自律実行。開発者は Operator API やエージェント SDK を活用して、ブラウジングを含む複雑なワークフローを構築可能

Atlas からの移行

Atlas ブラウザ向けに開発していたプラグインや拡張機能は、新しい配信チャネル (Chrome 拡張機能、デスクトップアプリ) に対応する形で移行が必要
