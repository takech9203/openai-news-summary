主な機能

🧠 Adaptive Thinking

動的推論時間配分、コスト効率の最適化。タスクの難易度に応じて推論リソースを自動調整します。

⚡ No-Think Mode

デフォルト推論なし、日常会話を高速化。シンプルなタスクでは推論をスキップし最速レスポンスを実現します。

💻 Codex ファミリー

Mini / Standard / Max の 3 段階コーディング性能。用途に応じた最適なモデルを選択できます。

🗄️ プロンプトキャッシュ 24h

長大システムプロンプトのコスト削減。24 時間のキャッシュ保持で繰り返し利用時のコストを大幅に削減します。

コードサンプル

Python GPT-5.1 API 呼び出し例 
from openai import OpenAI

client = OpenAI()

# Adaptive Thinking: 高度な推論
response = client.chat.completions.create(
    model="gpt-5.1",
    messages=[
        {"role": "user", "content": "Prove the sum of two even numbers is always even."}
    ],
    reasoning_effort="high"
)

# No-Think: 最速レスポンス (デフォルト)
response = client.chat.completions.create(
    model="gpt-5.1-instant",
    messages=[
        {"role": "user", "content": "Translate 'Hello' to Japanese."}
    ]
)
