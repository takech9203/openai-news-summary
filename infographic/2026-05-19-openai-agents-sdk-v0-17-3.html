Key Points

🔐

マウントポイント認証情報保護

サンドボックス環境のマウントポイントから認証情報が漏洩するリスクを排除。機密データへの不正アクセスを防止します。

🛡

パストラバーサル防止

相対パスを使用したディレクトリトラバーサル攻撃を検知し、ValueError を発生させることで不正なファイルアクセスをブロックします。

14 件のバグ修正

ガードレール例外のログ記録、None ガード処理、スキーマミューテーション防止など、広範な安定性改善を実施。

📝

ドキュメント改善

MCP サーバーの設定手順、ガードレールの使用例、サンドボックス構成のベストプラクティスなどを追加・更新。

Code Sample

Python v0.17.3 新機能の使用例 
from agents import Agent, OutputGuardrail
from agents.sandbox import SandboxConfig

# v0.17.3: ガードレール例外がログに記録される
agent = Agent(
    name="my_agent",
    output_guardrails=[my_guardrail],
)

# v0.17.3: 相対パスは拒否される (ValueError)
config = SandboxConfig(
    workspace_root="/absolute/path/only"
)
Bash インストールコマンド 
pip install --upgrade openai-agents==0.17.3

Developer Impact

プロダクション環境

セキュリティ修正により本番デプロイの安全性が向上。マウントポイント保護とパストラバーサル防止により、エージェントのサンドボックス環境で機密データが確実に保護されます。

Voice エージェント

カスタム音声処理の信頼性が改善。音声スプリッターのチャンク処理修正により、リアルタイム音声エージェントの応答品質と安定性が向上しました。

Vercel 統合

サンドボックス状態管理の効率化。Vercel 環境でのエージェントデプロイにおける状態の一貫性が改善され、ポートクエリの正規化により接続の安定性が向上しました。
