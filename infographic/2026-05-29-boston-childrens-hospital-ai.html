コードサンプル

Python 希少疾患診断 API 呼び出し例 
from openai import OpenAI

client = OpenAI()

# 患者の臨床データを匿名化して送信
patient_data = {
    "symptoms": ["発達遅延", "筋緊張低下", "特徴的顔貌"],
    "genetic_variants": [
        {"gene": "SCN1A", "variant": "c.1234A>G", "zygosity": "heterozygous"}
    ],
    "lab_results": {"lactate": "elevated", "CK": "normal"}
}

# Assistants API を使用した鑑別診断
assistant = client.beta.assistants.create(
    name="Rare Disease Diagnostic Assistant",
    instructions="""あなたは希少疾患の診断を支援する医療 AI アシスタントです。
    患者の症状、遺伝子バリアント、検査結果を分析し、
    可能性のある疾患候補を根拠とともに提示してください。
    OMIM、ClinVar のデータベースを参照してください。""",
    model="gpt-4o",
    tools=[{"type": "file_search"}]
)

# スレッドを作成して診断を実行
thread = client.beta.threads.create()
message = client.beta.threads.messages.create(
    thread_id=thread.id,
    role="user",
    content=f"以下の患者データを分析し鑑別診断を行ってください: {patient_data}"
)

# 実行して結果を取得
run = client.beta.threads.runs.create_and_poll(
    thread_id=thread.id,
    assistant_id=assistant.id
)

if run.status == "completed":
    messages = client.beta.threads.messages.list(thread_id=thread.id)
    diagnosis = messages.data[0].content[0].text.value
    print(f"鑑別診断結果:\n{diagnosis}")

ユースケース

🌍 診断の民主化

希少疾患の専門医が限られる地域でも、AI を活用することで高度な鑑別診断が可能に。地方の小児科医が専門施設と同等レベルの診断支援を受けられる環境を構築し、医療格差の解消に貢献

⏱️ 早期介入の実現

従来 5-7 年かかっていた「診断の旅 (diagnostic odyssey)」を数週間に短縮。早期に正確な診断を得ることで、適切な治療介入のタイミングを逃さず、患者の予後改善と家族の精神的負担軽減を実現

🔬 治療研究の加速

正確な診断が増えることで、希少疾患の患者コホートの形成が促進。臨床試験への参加機会が拡大し、新薬開発や遺伝子治療研究のパイプラインを加速
