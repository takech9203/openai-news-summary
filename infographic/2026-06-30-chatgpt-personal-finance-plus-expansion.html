ChatGPT パーソナルファイナンス機能が Plus ユーザーに拡大

2026-06-30

概要

💰

Plus プランへの拡大

5 月 15 日に Pro ($100/月) 限定で開始された個人向け金融機能が、Plus ($20/月) ユーザーにも一般提供 (GA) として拡大されました。

💳

Plaid によるセキュア接続

Plaid API を利用した安全な口座連携により、銀行口座、クレジットカード、投資口座をリアルタイムで接続。Intuit 連携も近日対応予定です。

🤖

AI 駆動の財務 Q&A

ユーザーの実際の財務データに基づいた AI による質問応答機能。支出データ、サブスクリプション追跡、今後の支払い予定、ポートフォリオパフォーマンスを分析します。

🎯

目標設定とメモリ連携

ChatGPT の Memory システムと連携した目標設定機能により、継続的な財務目標の追跡とパーソナライズされたアドバイスを提供します。

🔒

プライバシーファースト設計

データはいつでも削除可能。プライベートチャットでは財務データを使用せず、ユーザーが常にデータの管理権を保持します。

ユースケースと利用開始手順

自然言語での財務相談

「先月のサブスクリプション支出はいくら?」「節約するにはどうすればいい?」など、自然な日本語で自分の財務データについて質問できます。従来のルールベースの可視化ツール (Mint, Personal Capital) とは異なり、コンテキストを理解した動的なアドバイスが得られます。

支出パターンの分析

リアルタイム同期されたデータを基に、支出カテゴリの分類、異常な出費の検出、定期的な支払いの追跡を自動で行います。

目標ベースの財務計画

Memory と連携することで、「旅行資金を 3 ヶ月で貯める」「毎月の外食費を 20% 削減する」といった具体的な目標を設定し、進捗を追跡しながらアドバイスを受けられます。

競合との差別化

OpenAI は「一部の競合に追いつきつつある」段階ですが、ChatGPT のアプローチは自然言語インタラクション + コンテキスト認識型の動的アドバイスという点で、従来のルールベース可視化サービスと差別化されています。

利用開始の手順

  1. サイドバーから「Finances」を開く
  2. 「@Finances, connect my accounts」と入力する
  3. Plaid 経由でアカウントを接続する
  4. ダッシュボードで支出 / 投資状況を確認する
  5. 自然言語で財務に関する質問を行う

プライバシー保護

  • データはいつでも削除可能
  • 個人金融情報は Memory から除外設定が可能
  • プライベートチャットでは財務データを使用しない
  • ユーザーが常に「自分のデータを管理」している状態を維持
