client = OpenAI()

response = client.responses.create(
    model="gpt-5",
    input="このプロジェクトのテストカバレッジを 80% まで引き上げて、"
          "不足しているユニットテストを生成してください。",
    tools=[
        {
            "type": "codex",
            "sandbox": {
                "environment": "project-workspace",
            }
        }
    ],
)

🔌 API の統合・簡素化

複数の API エンドポイントが統合され、単一のインターフェースから全機能にアクセス可能に。開発体験が大幅に向上。

🤖 エージェント型ワークフローの強化
