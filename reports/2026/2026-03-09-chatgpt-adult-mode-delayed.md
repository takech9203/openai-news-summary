# ChatGPT Adult Mode が再び延期: 新たなリリース日は未定

## 概要

OpenAI は ChatGPT の「Adult Mode」(アダルトモード) のロールアウトを再び延期した。これは 2 度目の延期であり、新たなリリース日は提示されていない。OpenAI の広報担当者は Axios に対し「大人を大人として扱うという原則は変わらないが、適切な体験を実現するにはさらに時間が必要」と述べている。

同機能は 2025 年 10 月に CEO Sam Altman が発表し、当初は 2025 年 12 月のリリースを目標としていた。しかし 12 月の期限は達成されず、2026 年 3 月時点で 2 度目の延期が確認された。延期の背景には、年齢確認システムの技術的課題に加え、OpenAI が知性の向上、パーソナリティの調整、より積極的な応答スタイルの 3 領域を優先していることがある。

## 主な内容

### 延期の経緯とタイムライン

ChatGPT Adult Mode の開発経緯は以下の通りである。

1. **2025 年 10 月:** CEO Sam Altman が Adult Mode の構想を発表。チャットボットがエロティカを含む成人向けコンテンツの生成に対応する機能として紹介された
2. **2025 年 12 月:** 当初のリリース目標時期。しかし期限を過ぎてもリリースされず、最初の延期となった
3. **2026 年 3 月:** 2 度目の延期が確認。OpenAI の広報担当者が Axios に対して延期を認め、TechCrunch も同様に報じた。新たなリリース日は示されていない

OpenAI の広報担当者は次のように述べている: "We still believe in the principle of treating adults like adults, but getting the experience right will take more time."

### OpenAI が優先する 3 つの開発領域

OpenAI は Adult Mode のリリースよりも、以下の 3 つの領域を優先して開発を進めている。

- **知性の向上 (Intelligence upgrades):** モデルの推論能力や回答精度の向上。数日前にロールアウトされた GPT-5.4 Thinking モデルは、複雑な推論タスクに特化したアップデートであり、この方向性を反映している
- **パーソナリティの調整 (Personality adjustments):** ChatGPT の対話における人格や応答トーンの改善
- **より積極的な応答スタイル (More proactive response style):** ユーザーの意図を先読みし、能動的に情報を提供する応答スタイルへの転換

これらの優先事項は、Adult Mode よりも多くのユーザーに直接的な影響を与える基盤的な改善であり、OpenAI がプロダクト全体の品質向上を重視していることを示している。

### 年齢確認システムの技術的課題

延期のもう一つの大きな要因は、年齢確認 (Age Verification) システムの技術的課題である。初期テストにおいて、年齢検出システムに問題が発見されたことが報じられている。

成人向けコンテンツを提供するためには、未成年ユーザーを確実に排除する堅牢な年齢確認メカニズムが不可欠である。しかし、オンラインでの年齢確認は技術的に困難な課題であり、プライバシーの保護と正確な年齢判定の両立が求められる。OpenAI はこの検証システムの改善を延期の理由の一つとして挙げている。

### AI 安全性をめぐる社会的議論との関連

Adult Mode の延期は、AI の安全性と倫理をめぐる社会的議論が激化する時期に重なっている。

- **Pentagon との提携:** OpenAI は米国国防総省との提携を進めており、AI の軍事利用に関する議論を呼んでいる
- **社内の意見対立:** OpenAI 内部では方針をめぐる意見の相違が表面化しており、ロボティクス部門の責任者 Kalinowski 氏が Pentagon との取引発表を受けて辞任した
- **幹部の離脱:** 上級社員の離脱が続いており、組織の方向性に対する内部からの懸念が指摘されている

こうした状況の中で、成人向けコンテンツという議論を呼びやすい機能のリリースを急ぐことは、OpenAI にとって更なるリスクとなり得る。延期の判断には、技術的な課題だけでなく、こうした社会的・組織的な背景も影響している可能性がある。

## 開発者への影響

- **API への直接的な影響は限定的:** Adult Mode は主に ChatGPT の消費者向け機能であり、API を利用する開発者への直接的な影響は現時点では限定的である
- **コンテンツポリシーの方向性:** Adult Mode が最終的にリリースされた場合、API のコンテンツポリシーやモデレーション設定にも変更が波及する可能性がある。開発者は今後のポリシー更新に注意を払う必要がある
- **年齢確認の技術動向:** OpenAI が開発する年齢確認システムは、将来的に API を通じて開発者にも提供される可能性があり、年齢制限付きコンテンツを扱うアプリケーション開発に影響し得る
- **Moderation API の進化:** 成人向けコンテンツのフィルタリングや分類に関する技術は、Moderation API の機能拡張につながる可能性がある
