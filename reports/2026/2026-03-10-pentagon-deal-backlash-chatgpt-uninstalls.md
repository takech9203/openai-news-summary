## 概要

OpenAI の Pentagon (米国防総省) との契約発表を受け、米国で ChatGPT のアンインストール数が前日比 295% 急増する事態が発生した。消費者の間で AI の軍事利用に対する懸念が急速に拡大し、OpenAI CEO の Sam Altman は社内向けに弁明を行い、反発を「本当に辛い (really painful)」と表現した。

この騒動は AI 業界の競争環境にも大きな影響を与えている。Pentagon の契約条件に含まれる監視・自律型兵器関連の条項を拒否した Anthropic の Claude が、米国 App Store で無料アプリランキング 1 位に急浮上した。一方、Pentagon は Anthropic を「サプライチェーンリスク」に指定し、Trump 大統領は連邦機関に対して 6 か月以内に Anthropic 技術を段階的に廃止するよう命令した。AI 企業の軍事利用をめぐる社会的議論が、消費者行動と市場競争を大きく動かしている。

## 主な内容

### ChatGPT アンインストール急増の実態

Pentagon 契約の発表後、米国における ChatGPT のアンインストール数は前日比 295% という記録的な急増を見せた。hi-Tech.ua の報道によれば、ChatGPT の削除数は 3 倍に達したとされる。この動きは、AI の軍事利用に対する消費者の強い拒否反応を示している。

ユーザーが ChatGPT を離れる主な理由として、以下の懸念が挙げられている。

- **軍事監視への懸念:** AI が国内外の監視活動に利用される可能性
- **自律型兵器への懸念:** AI が人間の判断を介さずに殺傷力を持つ兵器に組み込まれる可能性
- **企業倫理への疑問:** 営利転換に続く軍事契約が、OpenAI の設立理念からの乖離を象徴

### Sam Altman の社内向け弁明

2026 年 3 月 9 日から 10 日にかけて、Sam Altman は OpenAI 社員に対して Pentagon との協力を弁護する声明を発表した (Mashable、MSN 報道)。

- **反発への率直な認識:** Altman は今回の反発を「really painful (本当に辛い)」と表現
- **運用上の責任分担:** Pentagon が AI の運用上の意思決定を管理し、OpenAI は技術提供に専念するという立場を説明
- **レッドラインの設定:** OpenAI は「国内監視には使用しない」「自律型兵器には使用しない」という明確な一線を維持していると強調

しかし、この弁明にもかかわらず社内の動揺は収まらず、3 月 8 日にはロボティクス部門責任者の Caitlin Kalinowski を含む複数の著名な社員が退職を表明している。

### Anthropic Claude の急成長と競争環境の変化

OpenAI の Pentagon 契約に対する反発は、競合他社にとって追い風となった。特に Anthropic が大きな恩恵を受けている。

- **App Store ランキング急上昇:** Anthropic の Claude が米国 App Store の無料アプリランキングで 1 位を獲得し、複数の国でも上位にランクイン
- **倫理的差別化の成功:** Anthropic は Pentagon の契約条件に含まれる監視・自律型兵器関連の条項を拒否しており、この姿勢が消費者の支持を集めた
- **政府からの圧力:** Pentagon は Anthropic を「サプライチェーンリスク」に指定。Trump 大統領は連邦機関に対し、6 か月以内に Anthropic 技術の段階的廃止を命令

この状況は、AI 企業の倫理的立場が直接的にユーザー獲得競争に影響する時代に入ったことを示している。

### AI 企業と軍事利用をめぐる社会的議論

今回の騒動は、AI 業界全体に対する社会的議論を活発化させている。

- **消費者の投票行動:** ユーザーがアプリのインストール・アンインストールという行動で企業の方針に対する賛否を表明する「ウォレット投票」が現実化
- **企業の倫理的選択:** AI 企業が軍事契約を受けるか否かが、ブランド価値と市場シェアに直結する構図が鮮明化
- **規制と市場の緊張:** 政府が特定の AI 企業を排除する動きは、自由競争と国家安全保障のバランスをめぐる新たな課題を提起

## 開発者への影響

### プラットフォーム選択の再評価

今回の騒動は、開発者やスタートアップが利用する AI プラットフォームの選択にも影響を及ぼしている。

- **マルチプロバイダー戦略の重要性:** 特定の AI プロバイダーへの依存リスクが顕在化した。OpenAI API に依存するアプリケーションは、ユーザー離れの影響を間接的に受ける可能性がある
- **Anthropic API への関心増加:** Claude の App Store ランキング 1 位獲得に伴い、Anthropic API への開発者の関心も高まっている
- **エンドユーザーの価値観への配慮:** 開発者は、利用する AI プロバイダーの企業方針がエンドユーザーの信頼に影響することを意識する必要がある

### 連邦政府案件への影響

- **Anthropic 技術の段階的廃止命令:** 連邦政府向けシステムで Anthropic API を利用している開発者は、6 か月以内の移行計画を策定する必要が生じる
- **OpenAI への集中リスク:** 政府案件における AI プロバイダーの選択肢が狭まることで、ベンダーロックインのリスクが増大
- **コンプライアンス要件の変化:** 軍事利用に関連するコンプライアンス要件が、民間向け API にも波及する可能性がある

### ブランドリスクの考慮

- 自社アプリケーションが「Powered by OpenAI」を前面に出している場合、ユーザーの反感がアプリ自体にも波及するリスクがある
- AI プロバイダーの選択を抽象化レイヤーで隠蔽し、柔軟に切り替え可能なアーキテクチャの採用が推奨される
